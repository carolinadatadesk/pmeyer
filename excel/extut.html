Online Excel Tutorials
Brought to you by ATN
ATN offers three online, interactive tutorials covering all aspects of Excel operations.  Accessing the programs requires use of CBT Player software.  This software has been loaded onto all of the computers in rooms 141 and 142.  To access the tutorials, go to http://cbt.unc.edu/cbt/curicula/custom.htm#Microsoft
Office97ENDUSER, click the link to the appropriate program and begin.

The tutorials for Excel in the Microsoft Office 97 library include:

  • Hands-on: Formatting text 
  • Creating formulas 
  • Hands-on: Entering a formula 
  • Automatic features and filling formulas 
  • Hands-on: Using AutoSum 
  • Date formulas 
  • Hands-on: Using a date formula 

Course Aim

To familiarize users with the Excel 97 environment and to enable them to perform basic spreadsheet operations

  • format rows, columns, and numbers 
  • create and use formulas

General Information
